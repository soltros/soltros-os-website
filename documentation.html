2Switch to SoltrOS Server

3Access Cockpit Web Console

Installation Methods

1ISO Installation (Recommended)

Download the official ISO and flash it to a USB device:

  • Bleeding-edge: Latest packages from Rawhide
  • Stable/LTS: Pinned to 6-month Fedora cycle

Once installed, SoltrOS rolls updates forever - no need to swap ISOs.

2Bootc Switch from Existing System

Switch from Fedora Silverblue/Kinoite/Bazzite/Bluefin:

sudo bootc switch ghcr.io/soltros/soltros-os:latest
sudo systemctl reboot

Or for the LTS version:

sudo bootc switch ghcr.io/soltros/soltros-os_lts:latest
sudo systemctl reboot

Important Notice

If you experience bootc issues due to policy.json problems, run these commands:

# Download the public key
sudo mkdir -p /etc/pki/containers
sudo curl -L https://github.com/soltros/Soltros-OS/raw/main/soltros.pub -o /etc/pki/containers/soltros.pub

# Download the secure policy
sudo mkdir -p /etc/containers
sudo curl -L https://github.com/soltros/Soltros-OS/raw/main/resources/policy.json -o /etc/containers/policy.json

# Verify permissions
sudo chmod 644 /etc/pki/containers/soltros.pub
sudo chmod 644 /etc/containers/policy.json

Quintuple Package Management System

SoltrOS implements five complementary package systems that work together seamlessly:

Layer 1RPM-OSTree (System Foundation)

Purpose: Immutable base system with atomic updates

Use for: Core system components, kernel, drivers, essential services

sudo bootc switch ghcr.io/soltros/soltros-os:latest

Layer 2Flatpak (Desktop Applications)

Purpose: Sandboxed desktop applications

Use for: Gaming platforms, creative tools, productivity software

sh /usr/share/soltros/bin/helper.sh install-flatpaks

Layer 3Distrobox (Development Environments)

Purpose: Containerized development environments

Use for: Isolated development, access to any Linux distro's packages

sh /usr/share/soltros/bin/helper.sh setup-distrobox

Layer 4Nix (Declarative Package Management)

Purpose: Declarative configuration with 80,000+ packages

Use for: Development tools, multiple versions simultaneously

sh /usr/share/soltros/bin/helper.sh install-nix
nixmanager install firefox

Layer 5Homebrew (Universal Compatibility)

Purpose: Fallback for packages not in other systems

Use for: macOS-familiar tools, extensive package selection

sh /usr/share/soltros/bin/helper.sh install-homebrew
brew install <package>
