Introduction to SoltrOS

SoltrOS is an immutable Linux desktop distribution built on Fedora Core OS, designed for gamers and developers who want a stable, rolling-release system that just works.

What is SoltrOS?

SoltrOS combines the stability of immutable operating systems with the flexibility of multiple package management systems. Built on the quay.io/fedora-ostree-desktops/base-atomic image, it delivers a gaming-optimized desktop experience with comprehensive developer tools.

Immutable by Design: Your base system is read-only and updated atomically. If an update fails or causes issues, you can instantly roll back to the previous working state.

Quick Facts

  • Base: quay.io/fedora-ostree-desktops/base-atomic (Cosmic uses cosmic-atomic upstream)
  • Desktop: KDE Plasma, Gnome, or COSMIC
  • Package Managers: 5 complementary systems
  • Update Model: Rolling release with atomic updates

Installation

Download the ISO and install SoltrOS like any Fedora system.

Downloads:

  • Unstable ISO - Latest from Rawhide
  • LTS ISO - Stable 6-month cycle

Create Bootable USB

sudo dd if=soltros-os-latest.iso of=/dev/sdX bs=4M status=progress
sudo sync

First Boot Setup

Essential Setup Steps

1. Install Flatpak Applications

sh /usr/share/soltros/bin/helper.sh install-flatpaks

2. Install Homebrew

sh /usr/share/soltros/bin/helper.sh install-homebrew

3. Install Nix

sh /usr/share/soltros/bin/helper.sh install-nix

4. Setup Nixmanager

sh /usr/share/soltros/bin/helper.sh setup-nixmanager

Five Package Managers

SoltrOS provides five complementary package management systems, each designed for a specific purpose.

Why Five?

🔒 RPM-OSTree

Rock-solid, unchanging base that can't be tampered with or messed up.

📦 Flatpak

GUI apps, mainstream applications, and sandboxed safety.

🍺 Homebrew

CLI tools for developers.

❄️ Nix

Apps you can't get as Flatpaks. Try things quickly with nix-shell.

📦 Distrobox

AUR access, DEBs, safe tinkering in containers.

Real-World Examples

Try Microsoft Edge Temporarily

nix-shell -p microsoft-edge
# Use Edge, then exit - it's gone

Install a Rogue DEB Package

distrobox enter ubuntu
sudo dpkg -i package.deb

Access AUR Packages

distrobox enter arch
yay -S <package>
